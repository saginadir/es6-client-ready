# es6-client-ready
This is a ready app structure for writing es6 on the client side.

Using gulp, babel and browserify. 

Inspired By This Guide:

- http://jpsierens.com/use-es6-right-now/

## Dependencies

- NodeJS
- NPM

First install all npm packages 

```
npm install
```

and then run gulp

```
gulp es6
```
